# Microservices & Computer Vision

Ce dépôt regroupe **deux projets/aires distinctes** :
- une partie **Microservices**
- une partie **Computer Vision**

Chaque dossier est **indépendant** et contient son **propre `README.md`** avec les explications complètes (installation, exécution, configuration, etc.).  
Pour comprendre comment ça fonctionne, commence par ouvrir le README **dans le dossier qui t’intéresse**.

## Structure du dépôt

- `Microservices/`  
  Projet(s) autour d’une architecture microservices.  
  ➜ Voir `Microservices/README.md`

- `computer-vision/`  
  Projet(s) et expérimentations en vision par ordinateur.  
  ➜ Voir `computer-vision/README.md`

> Remarque : les deux parties ne partagent pas forcément les mêmes dépendances ni la même façon de lancer le code.

## Comment naviguer rapidement

1. Choisis le dossier (`Microservices/` ou `computer-vision/`)
2. Lis le `README.md` du dossier
3. Suis les étapes “Prerequisites / Installation / Run”

## Objectif

Ce dépôt sert à regrouper mes travaux personnels (apprentissage + projets) autour de :
- **Microservices** (patterns, services, communication, déploiement, etc.)
- **Computer Vision** (traitement d’images, modèles, pipelines, etc.)
